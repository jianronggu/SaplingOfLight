## 🎮 OPTIONAL: PLAY WITH IT

- **Click & drag** = Rotate the object
- **Scroll** = Zoom in/out
- **Wait 4 seconds** = Title disappears
- **Move mouse** = Title comes back

---

## ✏️ OPTIONAL: CUSTOMIZE

Want to change the title?

1. Find this file in your editor:
   ```
   app/(scenes)/gallery/page.tsx
   ```

2. Find this line (near the top):
   ```
   title="Floating Crystal"
   ```

3. Change it to anything:
   ```
   title="My Amazing Art"
   ```

4. Save (Ctrl+S)

5. Look at your browser = Changed! ✨

---

## 🎨 OPTIONAL: ADD YOUR 3D MODEL

1. Download a .glb file (from Sketchfab or create one)

2. Copy it to this folder:
   ```
   public/models/
   ```

3. Find this line in gallery/page.tsx:
   ```
   modelPath="/models/Floating_Crystal.glb"
   ```

4. Change the filename to yours:
   ```
   modelPath="/models/your-file-name.glb"
   ```

5. Save

6. Look at browser = Your model appears! 🎨
