## Troubleshooting Quick Links

| Problem | Solution |
|---------|----------|
| Blank page | F12 → Console → Check errors |
| Model won't load | Check file in public/models/, check filename spelling |
| Server won't start | npm install --legacy-peer-deps, then npm run dev |
| Changes not showing | Save file, browser should auto-reload |
| Port 3000 in use | Kill process using port, restart server |
